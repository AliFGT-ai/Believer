What does the API has?

The API has a lot (and with the time will increase this stuffs) of things that the muslim need it for free, like:

  • Days Prayers
  • Monajats
  • Duaas
  • Visits (Ziyarat)
  • Prayers
  • etc...

HINT

We suggest you to use a JSON viewer extension, like Seven JSON Viewer

Duaa

Get more than 20+ duaa

Usage:

believer-api.alifgt.repl.co/duaa/

To get the duaa from her ID:

believer-api.alifgt.repl.co/duaa/[ID] believer-api.alifgt.repl.co/duaa/1 believer-api.alifgt.repl.co/duaa/2 believer-api.alifgt.repl.co/duaa/....

To get the Day Prayer from her ID:

To get day works:

believer-api.alifgt.repl.co/days-prayers/works/

To get day works by ID:
