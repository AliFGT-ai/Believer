What does the API has?

The API has a lot (and with the time will increase this stuffs) of things that the muslim need it for free, like:

  • Days Prayers
  • Monajats
  • Duaas
  • Visits (Ziyarat)
  • Prayers
  • etc...

HINT

We suggest you to use a JSON viewer extension, like Seven JSON Viewer

Duaa

Get more than 20+ duaa

Usage:

AAAAA/duaa/

To get the duaa from her ID:

AAAAAA/duaa/[ID] AAAAAA/duaa/1 AAAAAA/duaa/2 AAAAAA/duaa/....

Monajat

Get monajats

Usage:

AAAAA/monajat/

To get the monajat from her ID:

AAAAAA/monajat/[ID] AAAAAA/monajat/1 AAAAAA/monajat/2 AAAAAA/monajat/....

Days Prayers
